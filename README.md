##  Opis projekta

Ovaj projekat predstavlja web aplikaciju za upravljanje korisnicima (Users) koristeći:

- PHP (backend logika)
- MySQL (baza podataka)
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

Aplikacija implementira kompletan CRUD sistem:

- ✅ Create – Dodavanje novog korisnika
- ✅ Read – Prikaz svih korisnika u tabeli
- ✅ Update – Izmjena podataka korisnika
- ✅ Delete – Brisanje korisnika iz baze

---

## Funkcionalnosti

- Dodavanje korisnika (ime, prezime, email)
- Prikaz korisnika u tabeli
- Edit dugme za izmjenu podataka
- Delete dugme sa SweetAlert potvrdom
- Server-side paginacija
- JavaScript validacija forme
- Pretraga korisnika
- Responsive dizajn (Bootstrap)
